# Radar Vital

Contactless heart-rate / respiratory-rate monitoring with a Seeed MR60BHA2
60 GHz mmWave radar — a BS Electronics Engineering thesis project by
**Lemuel Blaya, Angelo Diaz, and Blessie Mugat**, University of Mindanao.
