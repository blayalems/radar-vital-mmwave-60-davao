Dashboard settings

Keep live mode honest. Demo mode is explicit, not silent.

Source mode

Demo modeUse generated sample telemetry instead of the trainer JSON feed. Intended for UI demos only.
Auto-enable demo on disconnectIf live polling fails repeatedly, switch to demo automatically. Default is off.

Disconnect behavior

Freeze on stale live dataWhen live data disappears, keep the last real payload visible and mark the dashboard stale instead of simulating new activity.

Appearance

ThemeChoose a fixed theme or follow the system setting.
DensityComfortable leaves more breathing room; compact fits more telemetry on screen.

Haptic feedback

Haptics mode Subtle vibration on session start, stop, export, copy, and destructive confirmations. Visual feedback remains the primary signal.

KPI alert thresholds

Changes apply immediately to live KPI alerts and persist across sessions.

Audio alerts

Enable audio alertsPlay a brief tone when HR or RR crosses a KPI threshold, or when the connection drops.
Voice announcementsUse browser speech synthesis for eyes-busy alert callouts.
Volume 70%

Live rendering

Waveform windowCap live chart history to prevent long-session redraw stalls.
Live charts keep 60 seconds, capped at 3600 points.
